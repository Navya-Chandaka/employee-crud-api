# Employee CRUD API

This project is a Spring Boot REST API for managing employee data with CRUD operations.

## Features
- Create Employee
- Get All Employees
- Update Employee
- Delete Employee

## Technologies Used
- Java
- Spring Boot
- Maven
- JPA
- REST API

## API Endpoints
POST /employees → Create employee  
GET /employees → Get all employees  
PUT /employees/{id} → Update employee  
DELETE /employees/{id} → Delete employee
